Computer aided design and manufacturing (CAD&M) Ops

The focus of this op is to muster and / or develop expertize in Electronic design automation (EDA) softwares for fabless semiconductor development as well as in 2D and 3D CAD softwares for precision engineered systems design.

This would be handy for quickly developing system prototypes and providing workable blueprints and core IP for semicondutor production.
