Data Ops

The usual troupe about data is usally along the line, 'data is the new oil'. Usually this is stated as a testament to it value in revealing actionable insight for both good and bad purposes. But this insight does introduce efficencies about systems from which there were collected.

The data ops unit is focused on data warehousing and analytics for creating models for better understanding as well as to seek out opportunities to make systems much efficient.
